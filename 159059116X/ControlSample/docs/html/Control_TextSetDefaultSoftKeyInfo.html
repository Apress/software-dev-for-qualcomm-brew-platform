void Control_TextSetDefaultSoftKeyInfo

( ITextCtl* pITextCtl,
  IMenuCtl* pIMenuCtl,
  uint16 okIcon,
  uint16 okString,
  PFNSTATEEVENT okState,
  uint16 cancelIcon,
  uint16 cancelString,
  PFNSTATEEVENT cancelState )

Sets the default soft key info for a text control
Sets the default menu items for a soft key control.
Parameters:
ITextCtl - *pITextCtl: target text control
IMenuCtl - *pIMenuCtl: target soft menu control
uint16 - okIcon: resource id of ok icon in menu
uint16 - okString: resource id of ok string in menu
uint16 - okState: state to enter when OK is pressed
uint16 - cancelIcon: resource id of cancel icon in menu
uint16 - cancelString: resource id of ok string in menu
uint16 - cancelState: state to enter when cancel is pressed
Returns:
nothing
