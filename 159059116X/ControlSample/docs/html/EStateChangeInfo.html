typedef enum EStateChangeInfo

State transition extra information.
StateChangeInfo_push
Nothing particular is known
StateChangeInfo_CLR
User pressed CLR key
StateChangeInfo_SELECT
User pressed SELECT key
Each state function is given one of these values in a bit field in the wParam during a state transition to indicate the reason for the state change.
