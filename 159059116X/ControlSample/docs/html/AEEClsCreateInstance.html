int AEEClsCreateInstance

( AEECLSID clsID, IShell* pIShell,
  IModule* pIModule, void** ppObj )

Create an instance of this class.

Documentation

Create an instance of this class. This constructor is invoked by the BREW shell with the applet is launched.

Parameters:
AEECLSID - clsID: class ID of the class being requested
IShell - *pIShell: a pointer to the BREW shell
IModule - *po: a pointer to the current module
void - **ppObj: a pointer to the created applet
Returns:
AEE_SUCCESS on success, with the applet in *pobj.
