typedef enum EStateChangeCause

State transitions.
StateChange_push
A new state is being pushed
StateChange_pop
The current state is being popped
StateChange_suspend
The system has sent an EVT_SUSPEND
StateChange_resume
The system has sent an EVT_RESUME
StateChange_causemask
The state change cause mask
Each state function is given one of these enums in a bit field in the wParam during a state transition to indicate the reason for the state change.
